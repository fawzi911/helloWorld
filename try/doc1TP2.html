  1. Les liens

    1. Les liens internes

Le langage HTML permet d'insérer des liens vers une autre partie du même document. Cela peut permettre par exemple de créer une table des matières en début de page, ou un renvoi au début en fin de page. La cible d'un lien interne est repérée par un identifiant du type id = "debut".

Retour haut de page

    1. Les liens externes

Le langage HTML permet d'insérer des liens vers une autre page web. La cible d'un lien externe est repérée par son adresse, comme dans la barre d'adresse du navigateur.

Un premier exemple de lien hypertexte qui s’ouvre dans un nouvel onglet vers le site de l’Institut Supérieur d’Informatique et Multimédia de Sfax : ISIMS

Un deuxième exemple de lien hypertexte qui s’ouvre dans une nouvelle fenêtre de navigateur vers le site de validation d’une page Web : W3C

  1. Insertion d'images

Pour insérer une image dans une page HTML, il faut indiquer une adresse qui permet au navigateur de trouver l'endroit où l'image est stockée.
